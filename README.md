# McDonald's Ordering Optimizer (麥當勞點餐優化器)

A high-performance web application designed to find the most cost-effective combination for McDonald's orders using dynamic programming. Specifically tuned for the 2026 Taiwan market.

![Premium UI](C:/Users/Aiersenlan_lab_PC/.gemini/antigravity/brain/5a0f071d-b6db-428a-af34-19b89eee5408/card_fix_final_1773342539614.png)

## ✨ Features

- **🏆 Dynamic Optimizer**: Uses memoized state-space search to guarantee the lowest possible price for any target cart.
- **🏷️ 2026 Promo Support**: Deep integration with current promotional schemes:
  - Sweetheart Card (甜心卡)
  - 1+1=$50 / 1+1=$69 Star deals
  - BOGO (買一送一) Coupons
  - App-exclusive discounts
- **🔥 Turbo Engine**: Optimized calculation logic responding in <10ms even for large orders.
- **🥗 Nutrition Tracking**: Real-time calorie (kcal) summation for both selected items and optimal strategies.
- **🌓 Dual Themes**: Premium dark and light mode support with modern aesthetics.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Algorithm**: State-space search with memoization
- **Styling**: Modern dark-themed CSS with glassmorphism effects

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/Aiersenlan/mcdonalds-optimizer.git
   ```
2. Open `index.html` in any modern web browser.
3. Start adding items to find your best saving strategy!
